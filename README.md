# 夏杰语音【中国大陆官方版】

#### 介绍
夏杰语音，官方免费版，适用于中国大陆地区。如果是其他国际区域，请下载国际版本。

该版本为高sdk版本，targetsdk35，支持安卓4.4到安卓15

低sdk版本【targetsdk22】，可以到当贝市场下载。

可以安装"AI智能管家"，自动开启相关权限和设置。

#### 使用说明
基本上安装就可以直接用，用户什么都不用管。
如果用不了，注意几个地方

1）打开语音页面，使用说明，尽量开一下权限。

2）如果无法开启权限，可以将夏杰语音设置为默认输入法。
